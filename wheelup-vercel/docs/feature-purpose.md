# wheelsUp 機能一覧 — 目的・背景・解決する問題

> ポジショニング: 大規模な業務システムではなく、**面談後の振り返りと知識習得のための断片的なツール群**。
> 既存業務（Pipedrive入力 → 面談 → フォロー）はそのまま。面談の「後」にワンアクションだけ追加することで、自然にスキルが上がる仕組み。

---

## 全体コンセプト

| 項目 | 内容 |
|------|------|
| 誰の | キャリアコンサルタント（建築技術者専門） |
| どんな悩み | リーダーが同席しないと質の高い面談ができない。自分の面談の何が良くて何が悪いか分からない |
| どうやって | 面談録音をアップするだけでAIが自動採点し、リーダーの面談パターンと比較してフィードバックする |
| どんな効果 | 2人体制→1人体制へ。リーダーの暗黙知が形式知になり、各自が自走できる |

---

## 機能一覧

### 1. 面談の自動採点

| 項目 | 内容 |
|------|------|
| **何をするか** | 面談の録音 or テキストをアップすると、AIが5軸×10点で自動採点する |
| **なぜ必要か** | 面談の良し悪しを「感覚」ではなく「数字」で把握しないと改善が回らない。しかし人間が毎回フィードバックを書くのは現実的でない |
| **背景にある課題** | リーダーが「あの面談よかったね」と言っても、具体的に何が良かったか伝わらない。コンサルタント側も「何を直せばいいか」が曖昧なまま次の面談に臨んでいる |
| **業務への影響** | 録音をアップするだけ。それ以外の操作は不要（自動採点）。結果は通知で届く |
| **期待する効果** | 面談1件あたり30秒の追加作業で、毎回具体的な改善ポイントが手に入る |

#### 採点5軸の設計意図

| 軸 | 何を見ているか | なぜこの軸か |
|----|---------------|------------|
| ニーズ把握 | 候補者の本音・課題を引き出せたか | ここが弱いとミスマッチ推薦→途中辞退に直結する |
| 提案力 | 具体的な求人を「なぜマッチするか」付きで提示できたか | 「とりあえず紹介」では候補者の信頼を失う |
| 信頼構築 | 業界知識を示し、専門家として認められたか | 建築業界は資格・現場経験の文脈が分かるかで候補者の心の開き方が全く違う |
| クロージング | 次のアクションを期限付きで握れたか | ここが弱いと「検討します」で止まり、他社に流れる |
| 情報収集 | 他社状況・温度感・決裁者情報を聞き出せたか | 企業に提案する際のカードが足りないと推薦が通らない |

---

### 2. 採点根拠の引用表示

| 項目 | 内容 |
|------|------|
| **何をするか** | 各スコアに対して、面談記録からの具体的な発言引用を根拠として提示する |
| **なぜ必要か** | 「AIに7点と言われた」だけでは納得感がない。人は「事実」を見せられないと行動を変えない |
| **背景にある課題** | 過去の教育ツール導入で「スコアが信用できない」→使わなくなる→形骸化、というパターンが多い |
| **業務への影響** | 追加作業なし。採点結果の中に自動表示される |
| **期待する効果** | 「確かにこの発言は弱かった」と自分で気づける。納得があるから次に活かせる |

---

### 3. リーダー面談との比較

| 項目 | 内容 |
|------|------|
| **何をするか** | リーダーの面談も同様に登録・採点し、コンサルタントのスコアと並べてレーダーチャートで比較する |
| **なぜ必要か** | 「何点だったか」だけでは目標が分からない。「リーダーと比べて何が足りないか」が分かって初めて方向性が定まる |
| **背景にある課題** | リーダーの「うまさ」は同席して見ないと伝わらない。しかし全商談に同席するのがボトルネックになっている |
| **業務への影響** | リーダーが自分の面談録音を月数件アップするだけ（それ以外は自動） |
| **期待する効果** | 「クロージングがリーダーより-3点」→具体的に何を変えればいいか明確になる |

---

### 4. 「リーダーならこうしてた」フィードバック

| 項目 | 内容 |
|------|------|
| **何をするか** | コンサルタントの面談を採点する際に、リーダーの面談パターンを参照し「この場面でリーダーならどう話したか」を具体的セリフ付きで生成する |
| **なぜ必要か** | 点数だけでは「次にどう言えばいいか」が分からない。具体的なセリフ例があって初めて再現できる |
| **背景にある課題** | リーダーの指導は「同席→その場でフィードバック」の形式。リーダーが不在だとフィードバック自体がゼロになる |
| **業務への影響** | 追加作業なし。採点結果の一部として自動表示 |
| **期待する効果** | 「次の面談ではこう言ってみよう」と具体的な行動変容に繋がる |

---

### 5. リーダープレイブック（自動生成）

| 項目 | 内容 |
|------|------|
| **何をするか** | リーダーの面談データ群からAIがパターンを抽出し、「状況別の対応マニュアル」を自動生成する |
| **なぜ必要か** | リーダーのノウハウは「頭の中」にしかない。退職・異動リスクもある。組織の資産として残す必要がある |
| **背景にある課題** | 「マニュアルを書いてください」と頼んでも、忙しいリーダーは書かない。録音をアップするだけで自動生成される仕組みなら負担ゼロ |
| **業務への影響** | リーダー: 面談録音アップのみ（既存の面談をするだけ）。コンサルタント: 面談前にプレイブックを参照（任意） |
| **期待する効果** | 「年収交渉でごねられたらどうする？」→プレイブックを開けば具体的対応が載っている |

---

### 6. 通知フィード

| 項目 | 内容 |
|------|------|
| **何をするか** | 採点完了時に「Bランク (35/50) — タップして根拠を確認」「提案力でリーダー超え！」などの一行通知を出す |
| **なぜ必要か** | 人は「結果が気になる」から見に来る。見に来る習慣ができれば定着する |
| **背景にある課題** | 業務ツールは「必要な時に開くもの」になると使われなくなる。「気になるから開くもの」にする必要がある |
| **業務への影響** | 追加作業なし。面談アップ後に自動で通知が届く |
| **期待する効果** | 「あ、採点きた。どうだったかな？」→クリック→振り返り→改善、のループが自然に回る |

## 設計思想まとめ: なぜこの構成で定着するか

```
既存業務フロー:
  Pipedrive記入 → 面談実施 → フォロー

wheelsUp で追加される作業:
  面談実施 → [録音アップ（30秒）] → 以降すべて自動

                    ┌──────────────┐
                    │  録音アップ   │ ← 唯一の追加作業
                    └──────┬───────┘
                           │ 自動
              ┌────────────┼────────────┐
              ▼            ▼            ▼
         文字起こし     自動採点     通知生成
              │            │            │
              ▼            ▼            ▼
         AI要約      スコア+根拠    「結果出たよ」
                      +リーダー比較    ↓
                      +改善セリフ   クリックしたくなる
                           │
                           ▼
                    振り返り → 次の面談で改善
```

### 定着の3条件

| 条件 | wheelsUpでの実現方法 |
|------|-------------------|
| **業務が増えない** | 録音アップ以外はすべて自動（採点・比較・通知生成） |
| **事実に基づいている** | スコアには面談記録からの引用が付く。「AIの感想」ではなく「あなたの発言の事実」 |
| **つい見たくなる** | 好奇心ギャップ通知（「あなたの〇〇はリーダーより+2高かった」）で自然にクリック |

---

## 補足: このツールが「教育システム」ではない理由

| 教育システム | wheelsUp |
|------------|----------|
| カリキュラムがある | ない。面談するたびに勝手にフィードバックが来る |
| 「学ぶ時間」を別に取る | 取らない。通知を見るだけ（10秒） |
| テストがある | ない。実際の面談が「テスト」 |
| 修了条件がある | ない。使い続ける限りスコアが蓄積される |

これは「教育」ではなく「鏡」。自分の面談を鏡に映すだけ。
鏡を見る習慣さえつけば、自分で直す。
